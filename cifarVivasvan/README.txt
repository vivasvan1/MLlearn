This is an CIFAR10 implementation for classification of 10 class.

The 10 classes are:

airplane
automobile
bird
cat
deer
dog
frog
horse
ship
truck

Data: is in the cifar-10-batches folder which i obtained from this link [https://www.cs.toronto.edu/~kriz/cifar.html]
instructions to unpickle the data are also provided in the above link.

This model has completed its training session with top-1 error of 6%.

To execute the code:

Run the following line in terminal:
python3 imple.py

P.S. :- this model can easily be trained a decent CPU (tested on intel i5 chip with 4GB dram).
